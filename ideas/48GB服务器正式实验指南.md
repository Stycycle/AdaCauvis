# 48GB 服务器正式实验指南

本指南用于在 `vGPU-48GB x 1` 服务器上运行正式对比实验。该服务器显存充足，不需要使用低显存配置。请使用原始 `640x640` 配置：

```text
configs/cauvis/cauvis_dinov2_dinohead_bs1x4_sdgod.py
```

不要使用：

```text
configs/cauvis/cauvis_dinov2_dinohead_bs1x1_sdgod_384_lowmem.py
```

也不要再加低显存参数：

```text
model.num_queries=100
model.backbone.cauvis_config.token_length=256
```

## 1. 训练 Baseline：固定 0.20 频域比例

作用：训练对照组。所有 Transformer 层都使用固定 `0.20` 低频保留比例。

```bash
XFORMERS_DISABLED=1 CUDA_VISIBLE_DEVICES=0 \
python tools/train.py \
  configs/cauvis/cauvis_dinov2_dinohead_bs1x4_sdgod.py \
  --amp \
  --work-dir work_dir/train_fixed_020_640 \
  --cfg-options \
  model.backbone.cauvis_config.min_low_freq_ratio=0.2 \
  model.backbone.cauvis_config.max_low_freq_ratio=0.2
```

产物：

```text
work_dir/train_fixed_020_640/epoch_12.pth
```

## 2. 训练改进版：Layer-Adaptive 频域比例

作用：训练改进组。默认使用浅层 `0.15` 到深层 `0.42` 的分层频域比例。

```bash
XFORMERS_DISABLED=1 CUDA_VISIBLE_DEVICES=0 \
python tools/train.py \
  configs/cauvis/cauvis_dinov2_dinohead_bs1x4_sdgod.py \
  --amp \
  --work-dir work_dir/train_layer_adaptive_640
```

产物：

```text
work_dir/train_layer_adaptive_640/epoch_12.pth
```

## 3. 测试 Baseline

作用：用 Baseline checkpoint 在完整 SDGOD 测试集上评估。

```bash
XFORMERS_DISABLED=1 CUDA_VISIBLE_DEVICES=0 \
python tools/test.py \
  configs/cauvis/cauvis_dinov2_dinohead_bs1x4_sdgod.py \
  work_dir/train_fixed_020_640/epoch_12.pth \
  --work-dir work_dir/test_fixed_020_640 \
  --cfg-options \
  model.backbone.cauvis_config.min_low_freq_ratio=0.2 \
  model.backbone.cauvis_config.max_low_freq_ratio=0.2
```

产物：

```text
work_dir/test_fixed_020_640/<timestamp>/<timestamp>.json
```

python tools/test.py \

## 5. 查找测试结果

作用：找到两个测试任务生成的 JSON 指标文件。

```bash
find work_dir/test_fixed_020_640 work_dir/test_layer_adaptive_640 \
  -name "*.json" -print
```

## 6. 比较指标

重点比较：

```text
pascal_voc/daytime_clear_mAP
pascal_voc/daytime_foggy_mAP
pascal_voc/dusk_rainy_mAP
pascal_voc/night_rainy_mAP
pascal_voc/Night-Sunny_mAP
pascal_voc/mean_mAP
```

整理成表：

```text
Method              Clear  Foggy  DuskRain  NightRain  NightSunny  Mean
Fixed 0.20
Layer-Adaptive
Delta
```

判断标准：

- `mean_mAP` 提升：整体有效。
- `dusk_rainy`、`night_rainy` 提升：频域分层对恶劣天气有效。
- `daytime_clear` 明显下降：滤波可能损伤源域，需要降低强度。
- 指标几乎不变：固定 schedule 作用有限，后续考虑 learnable ratio 或动态光照。

## 7. 注意事项

- 服务器只有 1 张 GPU，因此使用 `python tools/train.py`，不是 `tools/dist_train.sh ... 8`。
- `XFORMERS_DISABLED=1` 是为了避免 xFormers/CUDA 版本不匹配；48GB 显存足够时禁用 xFormers 更稳。
- Baseline 和 Layer-Adaptive 必须使用相同配置，只允许频域比例不同。
- 如果服务器 xFormers 与 CUDA 12.8 完全匹配，可以尝试去掉 `XFORMERS_DISABLED=1`，但第一次建议保留。
